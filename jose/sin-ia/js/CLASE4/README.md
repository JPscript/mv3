# Ejercicio: Interacción básica con HTML, CSS y JavaScript

## Objetivo
Afianzar los fundamentos de HTML, CSS y JavaScript mediante un ejercicio sencillo y breve.

## Enunciado
Senior Cat, la mascota experta de Ladrillos, quiere que construyas una pequeña página web donde pueda cambiar el color de fondo con un botón. El ejercicio consiste en crear:

1. Un archivo HTML con un botón y un texto de bienvenida.
2. Un archivo CSS para dar estilo al botón y al fondo.
3. Un archivo JavaScript que cambie el color de fondo al hacer clic en el botón.

### Requisitos mínimos
- Estructura HTML básica (doctype, head, body).
- Un botón con el texto "Cambiar color".
- Un texto de bienvenida para Senior Cat.
- CSS para que el botón sea visible y el fondo tenga color inicial.
- JavaScript para cambiar el color de fondo al pulsar el botón.

### Guía paso a paso
1. Crea el archivo `index.html` con la estructura básica.
2. Añade un botón y un texto de bienvenida.
3. Crea el archivo `styles.css` y vincúlalo en el HTML.
4. Estiliza el botón y el fondo.
5. Crea el archivo `script.js` y vincúlalo en el HTML.
6. Escribe el código JavaScript para cambiar el color de fondo al hacer clic.

### Solución explicada
- El HTML define la estructura y el botón.
- El CSS mejora la apariencia y define el color inicial.
- El JavaScript detecta el clic y cambia el color de fondo.

### Criterio de finalización
- Al hacer clic en el botón, el color de fondo cambia.
- El botón y el texto son visibles y estilizados.

### Errores comunes
- No vincular correctamente los archivos CSS y JS.
- Olvidar el selector del botón en JavaScript.
- No usar la estructura básica de HTML.

## Mini guía de trabajo en clase
1. Leer el enunciado y entender el objetivo (2 min).
2. Crear los archivos y la estructura básica (5 min).
3. Escribir el HTML y CSS (5 min).
4. Implementar el JavaScript (5 min).
5. Probar y verificar el funcionamiento (3 min).

Duración estimada: 20 minutos.
