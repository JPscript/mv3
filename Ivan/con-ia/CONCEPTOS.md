# CONCEPTOS

Anotar conceptos clave con:

- **Definición**: una descripción breve y precisa.
- **Metáfora**: una analogía que facilite recordar el concepto.

Ejemplo:

- **Concepto**: DOM
  - **Definición**: Document Object Model, representación en árbol del HTML.
  - **Metáfora**: el árbol genealógico del documento, donde cada nodo es un familiar.
