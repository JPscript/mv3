# Instalación y configuración rápida

## 1) NVM

1. Instalar `nvm` para Windows.
2. Verificar: `nvm version`.

## 2) Node.js

1. Instalar versión LTS: `nvm install 22`.
2. Activar versión: `nvm use 22`.
3. Verificar: `node -v` y `npm -v`.

## 3) PostgreSQL

1. Instalar PostgreSQL.
2. Crear usuario y contraseña de desarrollo.
3. Verificar acceso con `psql` o pgAdmin.

## 4) Extras recomendados

- `git` y cuenta de GitHub.
- VS Code + extensiones básicas (ESLint, Prettier, GitHub Copilot opcional).

## 5) Comprobación final

- Node y npm responden en terminal.
- PostgreSQL está levantado y permite conexión local.
