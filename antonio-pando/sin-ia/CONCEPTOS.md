# Conceptos (sin IA)

Documento vivo de conceptos clave aprendidos sin ayuda de IA.
Cada concepto debe incluir:
- Definición corta.
- Alegoría o metáfora para entenderlo mejor.

## Ejemplo inicial
- CSS Box Model: contenido, padding, borde y margen forman la caja de un elemento.
  Metáfora: como una caja de envío con objeto, relleno, cartón y espacio exterior.
- Etiqueta de html.
- Atributo de html.
- Varias etiquetas a, img, section, h1, h2...
