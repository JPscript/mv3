Estructura semántica — Ejercicio guiado

Introducción

En este ejercicio aprenderás a usar etiquetas semánticas como header, main, section y footer para organizar el contenido de una página.

Actividad

Crea una página con una cabecera clara, al menos dos secciones con títulos y un pie de página con información de contacto. Usa elementos semánticos en lugar de div cuando sea posible.
