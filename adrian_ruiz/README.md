# Carpeta personal de Adrián

Este directorio contiene el trabajo personal de Adrián Ruiz en el máster, siguiendo la estructura establecida en el repositorio:

- `con-ia/` ejercicios resueltos con apoyo de IA
- `sin-ia/` ejercicios sin ayuda de IA
- `bitacora/` apuntes de cada clase
- `requisitos/` instrucciones de instalación y configuración del entorno

Se incluyen carpetas vacías que llevan dentro `.gitkeep` para que Git las rastree.
