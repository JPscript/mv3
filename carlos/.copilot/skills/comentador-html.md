# Skill: Comentador HTML

## Descripción
Esta skill analiza un archivo HTML y añade comentarios explicativos sobre las principales etiquetas, como `<header>`, `<nav>`, `<main>`, `<section>`, `<form>`, etc.  
El objetivo es ayudar a comprender la estructura del documento y facilitar el aprendizaje.

---

## Funcionamiento
La skill recibe un archivo HTML como entrada y devuelve una versión comentada del mismo.  
Los comentarios se insertan justo encima de cada etiqueta relevante.
