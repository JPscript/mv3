# AGENTS.md

Documento resumen para enrutar instrucciones según la herramienta.

## Fuente principal por herramienta

- GitHub Copilot (VS Code/GitHub): usar [.github/copilot-instructions.md](.github/copilot-instructions.md).
- Claude Code: usar `CLAUDE.md` (si existe).
- Cursor: usar `.cursorrules` o `cursor/rules/*.mdc` (si existe).

## Política de mantenimiento

- Evitar duplicidad de reglas entre archivos.
- Mantener este archivo corto y de referencia.
- Actualizar aquí solo los punteros; las reglas operativas van en el archivo específico de cada herramienta.
