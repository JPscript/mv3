# Conceptos (con IA)

Documento vivo de conceptos clave aprendidos con apoyo de IA.
Cada concepto debe incluir:
- Definición corta.
- Alegoría o metáfora para entenderlo mejor.

## Ejemplo inicial
- HTML semántico: usar etiquetas con significado para ordenar el contenido.
  Metáfora: como etiquetar cajones en una cocina para encontrar todo más rápido.
