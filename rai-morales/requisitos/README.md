# Requisitos del entorno

En esta carpeta centralizamos la instalación y configuración base del curso.

## Qué configurar
- `nvm` (gestor de versiones de Node.js)
- `nodejs` y `npm`
- `postgresql`
- Otros necesarios: `git`, `vscode`, cliente SQL (opcional)

## Objetivo
Tener un entorno estable y repetible para trabajar en todas las clases y prácticas.
