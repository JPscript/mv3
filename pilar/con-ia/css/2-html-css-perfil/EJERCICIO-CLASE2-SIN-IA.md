# CLASE2 - Ejercicio único (sin IA)

Referencia: `jp/bitacora/CLASE2.md`
Carpeta: `jp/sin-ia/html/2-html-css-perfil`

## Objetivo
Crear una página de **perfil personal con zona de contacto** usando HTML básico y mejorarla con CSS básico.

## Parte 1 (HTML)
Crea un `index.html` que incluya, como mínimo:
- Estructura semántica: `header`, `nav`, `main`, `section`, `footer`.
- Jerarquía de títulos: `h1`, `h2`.
- Párrafos para presentación personal.
- Una lista de habilidades o intereses.
- Una imagen de perfil con `alt`.
- Enlaces (`a`) en navbar y en sección de contacto.
- Zona de contacto con `input` (nombre, email) y `button`.

## Parte 2 (CSS)
Crea un `styles.css` y aplica:
- Tipografía legible y tamaño coherente.
- Paleta simple de colores (fondo, texto, acento).
- Espaciado (márgenes y paddings) consistente.
- Navbar ordenada.
- Layout simple con `display: flex` en al menos una sección.
- Estilo de botones (hover opcional).

## Criterio de finalización
- Página funcional y ordenada.
- Se usan todos los elementos pedidos.
- El CSS mejora claramente la presentación sin romper la estructura.

## Entregables
- `index.html`
- `styles.css`
- Dudas (si hay) en `jp/sin-ia/html/DUDAS.md`
